# Modern Photo Studio Website

## Overview
A premium luxury photography studio website showcasing services, portfolio, and contact information. Built with HTML, CSS, and JavaScript for a modern, responsive experience.

## Features
- Responsive navigation with mobile toggle
- Hero section with cinematic background
- Service categories with icons
- Portfolio gallery with filters and lightbox
- Testimonials slider
- Location map and contact details
- Floating WhatsApp button and back‑to‑top button

## Getting Started
1. **Clone or copy the repository** to your local machine.
2. **Install dependencies** (if any) with:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
   The site will be available at `http://localhost:3000` (or the port configured by Vite).

## Project Structure
```
web/
├─ index.html          # Main HTML file
├─ index.css           # Custom stylesheet
├─ app.js              # JavaScript for interactivity
├─ README.md           # This documentation file
└─ assets/             # Images and media assets
```

## Customisation
- Update the meta tags in `index.html` to reflect your studio’s details.
- Replace placeholder images in the hero, portfolio, and footer sections with your own high‑resolution photos.
- Modify the service cards, statistics, and testimonials to match your offerings.

## Deployment
When ready to go live, build the production bundle:
```bash
npm run build
```
Upload the generated `dist/` folder to your web host.
